# Rss feed application
Basic rss feed app created with Nest JS.

## Installation

1. Install dependencies
    ```sh
    npm i
    ```

2. Copy content from **.env.example** to **.env** and populate the required parameters

    ```sh
    cp .env.example .env
    ```
3. Run database with docker 
    ```sh
    docker-compose-up
    ```
4. Run application
    ```sh
    npm run start:dev
    ```
